## 字典過濾功能 (`dict.txt`)
程式會讀取同目錄下的 `dict.txt` 進行自定義替換或忽略。

### 格式說明：
-`舊詞,替換詞`（中間用半角逗號隔開）

- **連結替換標籤**：
  - `[TWITCH_LINK],圖奇連結`
  - `[YOUTUBE_LINK],油兔連結`
  - `[OTHER_LINK],發了個連結`
- **忽略特定詞彙**（逗號後留空）：
  - `討厭的詞,`
- **一般詞彙替換**：
  - `777,溜溜溜`

## 啓用 Auto 加速
針對高人氣或洗屏頻繁的聊天室，程式內建智能流控系統：

- **運作邏輯**：當待播放留言堆積超過 **5 條** 時，程式會自動切換至「快速語速」。
- **自動恢復**：當留言消化完畢，排隊數掉回門檻以下，會自動恢復為「正常語速」。
- **優點**：確保語音進度不會落後直播畫面，同時兼顧平時的聽感舒適度。
